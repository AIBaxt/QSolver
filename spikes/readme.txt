This directory contains various spikes to gain required knowledge for building a proper quadsolver.
Within each directory contains a readme with specific program information including how to run.

Overview of directories:

BasicIO - A program for experimenting with basic input/output that reads a line from the user and spits it back out until
the user enters 'q' or "Q".

dblcmp - A program for experimenting with comparision of two double numbers. Does not take user input.

floatcmp - A program for experimenting with comparisons between two floating point numbers. Does not take user input.

IEEE - Various IEEE-FP tests.

cunit - A very simple test for asserting that two integers are equal or not equal.
